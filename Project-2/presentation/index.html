History of Loading Spinners

"A loading spinner aka a throbber is an animated graphical control element used to show that a computer program is performing an action in the background". This presentation shows the history of a loading spinner.

By Batul Bahrainwala

Presentation Summary


One of the early uses of a throbber occurred in the NCSA Mosaic web browser of the early 1990s. It featured an NCSA logo that animated while Mosaic downloaded a web page. Later on Netscape and Internet Explorer took on similar ideas. They intigrated the throbber into their logos. To make the browers simplar and more user friendly, web browsers intigrated their Loading spinners into the mouse. The first version was a counting hand. Different web browers then started creating there own variations of the same. Todays web browers have started creating their own loading spinners based on their branding and identity
